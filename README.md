# React PageFlip Example

Interactive demo of `react-pageflip`, showcasing smooth page turning in a React + Vite app.

## Live Demo

- https://phucvu1601-tech.github.io/react-pageflip-example/

## Highlights

- Realistic flipbook UI using `react-pageflip`
- Built with React Router v7
- Powered by Vite and deployed as a static GitHub Pages site
- Ideal as a learning reference or UI prototype

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173` to view the app.
